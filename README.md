# GURPSRandomNPCGEN
Random NPC generator for the GURPS roleplaying system.

## Installation

This project was made using visual studio community 2017.

## Usage

```c#
RandomNPC randNPC = new RandomNPC();
randNPC.GetRandomNPC();
```
